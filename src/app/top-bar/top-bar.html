Cremona CAD

Cremona CAD is a parametric drafting tool for luthiers. It generates precise, geometrically-accurate violin outlines and internal templates based on historical and modern making traditions. All drawing happens in the browser — no installation, no cloud account.

Available Recipes

  • Enrico Ceruti Violin — proportions derived from the Cremonese master's surviving instruments.
  • Kelly Violin — a contemporary American workshop model.
  • Beard Violin — a traditional pattern used in craft-school instruction.

Files

Projects are saved as plain .json files. Use Save to export the current recipe parameters and Load to restore a previous session. Files are recipe-specific; a Ceruti file cannot be loaded into the Kelly recipe.

Reference Images

Any recipe panel lets you overlay a reference photograph or scan on the draft canvas. Use the opacity and transform controls inside the recipe sidebar to align it with the generated draft.

Night / Day Mode

Toggle the ☾ / ☀ button in the top-left to switch between dark and light themes. Your preference is saved automatically.

Getting Started

  1. Pick a recipe

    A recipe is a set of proportional rules that describes a particular violin model. The active recipe is shown in the sidebar on the right.

  2. Adjust the parameters

    Each recipe exposes sliders and numeric inputs that control everything from body length to upper/lower bout widths and corner placement. The draft canvas updates in real time.

  3. Navigate the canvas

    Scroll to zoom in and out.
    Middle-click drag (or right-click drag) to pan.
    The draft is always centered on load.

  4. Overlay a reference image

    In the recipe sidebar, locate the Reference Image section. Upload any image (scan, photo, PDF export) and use the transform controls to scale and position it behind the draft lines.

  5. Save your work

    Click Save in the top bar to download a .json snapshot of all current parameters. Click Load to restore it in a future session.

  6. Start fresh

    Click New to reset all parameters to the recipe defaults.

Tips

  • The canvas updates continuously; there is no "Apply" step.
  • Save often — the browser does not auto-save between sessions.
  • Day/Night mode preference persists across sessions via localStorage.
