CremonaCad

CremonaCad is a free and open source application for designing violin family instruments!

Rather than a static trace, or a complex list of coordinates, CremonaCad defines instruments using a simple system of intersecting arcs. These patterns are historically informed by the drafting document found in the workshop of Enrico Ceruti, as well as the research of David Beard, Kevin Kelly, and François Denis.

Upload a historical instrument and use it as a reference, or go off the rails and create something totally unique! :D

Once your design is complete, CremonaCad can create moulds and templates for your specific instrument. These patterns can be exported either as PDFs for traditional hand-making, or as SVGs for computer assisted fabrication.

Tutorial

Overview

CremonaCad uses input controls on the right side of the screen to draw your instrument on the canvas in real time. The process is defined by a series of sections which build upon one another, drawing each part of the instrument step by step. Once all the sections are complete, you can generate moulds and templates for construction.

  • The canvas comes with pan and zoom controls to allow precise visualization of your design.
  • Reference images can be loaded and scaled using the Reference controls on the bottom right of the canvas.
  • The Axis controls on the bottom right of the canvas allow for custom grid settings.
  • Files can be saved and loaded using the New Save and Load buttons at the top right of the screen.

Base Measurements

This section sets the primary dimensions of the instrument, the setting of a reference image, as well as basic file properties.

  • CremonaCad comes with a set of predefined templates from historical instruments. Select one to study, or create a new design.
  • Use the Upload Reference Image button to add your own images to aid in your design process.
  • Use the primary dimensions section to define the height and width of your instrument. This is also useful for scaling your reference image.
  • For many measurements, ratios will be displayed to the right of the metric value. Many Cremonese instruments use simple integer ratios which define their arc segments.
  • Define rib and overhang values to create a total inset value. This inset will bound the inner path of the instrument.

Main Bouts

The main bouts define the upper and lower sections of the instrument's body. These will be the first arc segments you create.

  • The join arcs U0 and L0 (Upper and Lower) are used to connect the Vesica arcs U1 and L1.
  • Each bout has a width measurement which sets the total outer width of the instrument including the inset. These width values are often provided in online measurements of historical instruments, and are helpful in guiding your design.
  • Use the Viol Neck option to add a viola da gamba-style upper extension. These are of course common on double bass.

Corners

This section defines both the position of the corners and the arc segments that connect them to the main bouts.

  • XY controls exist for each corner placement. In general I have tried to avoid cartesian coordinates, but this allows for precise adjustments.
  • The Use Viol Corner option allows for the creation of viol-style corners, which only have one connecting arc.

Center Bout

The center bout is the narrowest section of the instrument, connecting the upper and lower bouts.

  • The Center Bout Width defines the total outer width at the narrowest point. When adjusting the C0 circle, this width will remain constant.
  • The Fit C0 to Bouts option automatically calculates C0 to cleanly intersect U2 and L2. This is based off of Kevin Kelly's four-circle method, while still allowing for manual adjustments.

Outer Path

The outer path section defines the button and corner arcs that complete the outline of the instrument.

  • Each corner is defined by at least two corner arcs and respective cutoff angles. The cutoff values define points at which a line is drawn across the outer corners.
  • Some violins employ more elaborate corners with double arcs, which you may employ on any corner.
  • The button is a small half-circle segment that is usually found on the backs of violin family instruments. You can define its dimensions here.

Mould

The mould section defines the internal blocks and templates used to construct the instrument. These blocks form the wooden structure around which the ribs are bent and glued.

  • Four blocks are defined: Upper, Upper Corner, Lower Corner, and Lower. Each block has height and width dimensions.
  • Corner block Pad values add extra material at each corner point. This can help with finding the optimal placing of the block against the mould.
  • If the mould is to be cut on a CNC, the Bit Diameter value prevent issues with right angles at critical glue points.
  • The default mould has clamping channels parallel to the edges. However, a simple box cutout can be specified as well.

Export

Once your design is complete, you can export various templates and patterns for construction. CremonaCad generates both SVG and PDF files.

About The Project

Hello there! y name is Andrew Argraves, a software engineer, cellist, and aspiring luthier living in New Haven, CT!

CremonaCad is a passion project initially developed as a tool to better understand the geometry of violin family instruments. However, as the project has developed over many months of late-night coding, I am beginning to see its potential as a design aid and educational tool for the maker community at large.

I intend to keep developing CremonaCad for some time, and hope to eventually add features for arching templates, f-holes, and neck design.

I believe strongly in open source software and community collaboration. If you are interested in collaborating on CremonaCad, feel free to reach out:
